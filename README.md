# 🛒 Sistema de Cadastro de Produtos em Java

Projeto desenvolvido em Java com foco em prática de Programação Orientada a Objetos (POO) e estruturas de dados.

## 🚀 Funcionalidades

* Cadastro de produtos
* Listagem de produtos
* Cálculo de preço com desconto
* Remoção de produtos
* Cálculo de média de preços

## 🧠 Tecnologias utilizadas

* Java
* ArrayList
* Scanner
* Estruturas de repetição (do-while)
* Estrutura de decisão (switch)

## 📚 Objetivo

Projeto criado com o objetivo de praticar lógica de programação e evoluir para desenvolvimento Java backend.
